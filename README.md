# MUTUAL-FUND-ANALYTICS
# Mutual Fund Analytics Platform

## Overview

This project was developed as part of the Bluestock Fintech Capstone Program.

The Mutual Fund Analytics Platform provides end-to-end analytics on Indian mutual funds using Python, SQLite, Jupyter Notebooks, and Power BI.

The project covers data ingestion, ETL processing, exploratory data analysis, performance analytics, advanced risk analytics, and interactive dashboards.

---

## Objectives

* Build an ETL pipeline for mutual fund data
* Perform exploratory data analysis
* Calculate fund performance metrics
* Develop advanced risk analytics
* Create investor analytics
* Build a fund recommendation engine
* Design an interactive Power BI dashboard
* Deliver business insights and recommendations

## Key Findings

* SIP inflows reached record highs during the study period.
* SBI Mutual Fund maintained the highest AUM.
* Top-ranked funds delivered superior risk-adjusted returns.
* Investor participation increased significantly across cohorts.
* Most SIP investors were identified as at-risk based on contribution gaps.
